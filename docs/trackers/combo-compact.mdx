# (Combo) Compact Tracker

<img src={useBaseUrl('/img/compact-tracker.png')} alt="Compact tracker" width="300" />

## Connectivity

The Abeeway Compact Tracker is available in two variants:

- **Compact Tracker**, which supports:
  - LoRaWAN connectivity
- **Combo Compact Tracker**, which supports:
  - Cellular (LTE-M, NB-IoT)
  - LoRaWAN connectivity

## Embedded sensors

- Accelerometer
- Multi-constellation GNSS
- Low-power (assisted) GPS
- Wi-Fi scanner
- BLE scanner
- Tamper sensors (tamper switch and hall-effect sensor)

## Firmware Features

### Connectivity

#### LoRaWAN connectivity features

- Class A operation
- Support for all major regional parameters (EU, AS, IN, AU, US)
- Over-the-Air Activation (OTA)
- Network monitoring (link check, reset on prolonged network loss)
- Multiple transmission strategies (configurable ADR, dual transmissions)
- Configurable confirmed uplinks
- Configurable heartbeat period
- Synchronization to network time

#### Cellular connectivity features (for the Combo variant)

- Support for NB-IoT and LTE-M
- Support for physical SIM and eSIM
- Support for PSM and eDRX modes
- Configurable heartbeat period
- Network time synchronization

### Geolocation

- Configurable multi-technology geolocation profiles
  - GNSS
  - Assisted GNSS (Low-Power GNSS / LP-GNSS)
  - BLE Scan
  - Wi-Fi Scan
- Configurable triggers for location updates:
  - Button press
  - Motion-start, Motion-end
  - Periodic triggers for in-static and in-motion states
  - SOS events
  - Additional event-based triggers
- BLE positioning with filtering options:
  - Beacon type filtering: Eddystone, Altbeacon, iBeacon
  - Beacon ID pattern matching
- Configurable geolocation timeouts

### SOS support

- Dedicated UI for triggering SOS messages
- Configurable geolocation behavior during SOS mode

### BLE inventory management/BLE scan collection

- Reporting of up to 20 BLE beacons with beacon-type and beacon-identifier filtering

### BLE beaconing

- Configurable beacon transmission (Eddystone, Altbeacon, iBeacon & Quuppa)

### BLE connectivity

- Single bonding / Passkey authentication support
- BLE firmware update over the air
- MCU firmware update over the air
- CLI over BLE

### Device management

- Command Line Interface (CLI) over USB or BLE connected console
- Event logging over CLI
- Accelerometer management (motion and shock detection)
- Temperature management (permanent storage of min/max values, actions when thresholds are reached)
- Battery management (primary vs. rechargeable batteries, remaining capacity measurement, and estimated consumption)
- Configuration parameter management (permanent storage, configuration files, and parameter identifier parsing)
- Firmware update over USB, BLE, and cellular network (FUOTA)
