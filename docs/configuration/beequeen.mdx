# <><img src={useBaseUrl('/img/beequeen-icon-small.png')} alt="BeeQueen logo" width="40" /> BeeQueen</>

[BeeQueen](https://tools.abeeway.io/beequeen) is a web-based configuration file editor that runs in Chrome.

It is also a useful way to get familiar with the main device features and configuration options.

To create a new configuration file, open BeeQueen in Chrome and select *"File -> New (default)"*.

Configuration parameters are grouped into categories such as *System*, *Core*, *Geoloc*, and *GNSS* as well as several others. These categories can be selected using the tabs at the top of the screen.

If you are new to the device configuration, a good starting point is:

- Start with the *Geoloc* group, which defines which geolocation technologies are used and when they are triggered.
- Then review the technology-specific groups such as *GNSS*, *LR11xx*, and *BLE Scan*.
- Finally, look at the connectivity-related groups such as *Network*, *LoRaWAN*, and *Cellular*.

When you finish editing the configuration file, select *"File -> Save as..."* to save it.

The saved configuration file can then be imported into the device using [BeeHive](https://tools.abeeway.io/beehive).
